How I created a web site.

  1. Create an account on git hub
  2. Create a public project, and include the readme.md when asked(important)
  3. Create a new file using the git hub UI, call it index.html, and put some start up html in there.
  4. Select settings
  5. click Pages in the Code and automation section, this takes you to, by way of example:
    https://github.com/mzallocco/testPages/settings/pages
    Ensure the following are specified:
    Build and deployment
    Source : Deploy from a branch
    Branch: main/root then save
Your URL is https://<userid>.github.io.<projectName>
e.g. https://mzallocco.github.io/testPages/
When you modify index.html, need to repeat 5.
For a more detailed description:
https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site
